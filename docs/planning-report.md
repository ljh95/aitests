## 핵심 방향성 의견 정리

### 포지셔닝 (대중성 분석가 제안, 전원 동의)
> **"AI한테 잘 물어보려면, 이 앱부터."**

- 타겟: 개발자 & 학습자 (일반 대중 확장은 현재 불가 — 마크다운 진입장벽)
- 차별점: "AI Copy" 3모드(Full/Chunks/Template) — 경쟁사에 없는 기능
- 위험: "복사가 더 편한 것"만으로는 부족, **양방향 AI 워크플로우** 필요

### 리텐션 (기획자 강조)
현재 리텐션 장치 **전무**. 다음 3가지 중 최소 1개 필요:
1. **스트릭 시스템** — 연속 작성일 표시 (Effort: 낮음, Impact: 높음)
2. **오래된 노트 리마인드** — 7일마다 과거 노트 1개 추천
3. **월간 대시보드** — 이번 달 작성 노트 수, 가장 많이 AI에 보낸 주제

### 수익화 (대중성 분석가)
- 현재 형태: 수익화 불가
- 가장 현실적 경로: **프리미엄 모델** (무료 100노트 / Pro $4.99/월 무제한)
- 장기: 교육기관 B2B, 팀 협업 플랜

## 추천 실행 로드맵

### Phase 1: 안정화 (1-2주)
- 모바일 비동기 초기화 버그 수정
- Android 백버튼 처리
- 자동 저장 상태 UI 표시
- Safe Area CSS 처리

### Phase 2: 핵심 기능 보강 (2-4주)
- 태그 시스템 UI (입력, 필터링)
- 커스텀 프롬프트 템플릿
- 코드 블록 Copy 버튼
- IndexedDB 마이그레이션 (용량 확장)

### Phase 3: 리텐션 & UX (2-4주)
- 스트릭 시스템
- 온보딩 튜토리얼
- 월간 대시보드
- 오래된 노트 리마인드

### Phase 4: 생태계 확장 (1-2개월)
- Google Drive 동기화
- Android 공유 인텐트
- 브라우저 확장
- AI 응답 저장 (양방향)

### Phase 5: 출시 준비 (2주)
- 앱 아이콘/스플래시 디자인
- 스크린샷 5-8장
- 개인정보보호정책
- Play Store 등록
